# Stitch UI SSOT (Nimbus Drive)

- 이 폴더는 **UI 구현의 참조 원천**입니다.
- 기본 마이크로카피는 **한국어(ko-KR)** 입니다.
- 영어(en-US)는 `en-us_preview/`의 언어 전환 프리뷰에서만 참조합니다.
- 페이지 코드는 **ui-kit 컴포넌트 조합만** 허용합니다(복붙 JSX 금지).
- 누락 여부는 `inventory/`의 Deliverables Inventory 보드로 확인합니다.
## HARD RULE
- 메뉴/설정 섹션 구조는 `docs/ui/IA_NAV_SSOT.md`가 정본이다.
- 모든 UI 텍스트는 `docs/ui/COPY_KEYS_SSOT.md` 키를 사용하며 하드코딩 금지.
